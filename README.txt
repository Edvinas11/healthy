Healthy

ABOUT
Healthy is a command line program for calculating ideal daily calories intake and more. Also, this program creates a weekly list of various recommended products.

USAGE
$ python main.py 

After the program starts, user is obliged to type his name. Then, 4 questions are given to the user. 
User must answer the given questions in order to run successfully for the program.
After this process, user will see daily calories, carbs, protein and fat ideal intake. 
To add to his, user will also see his list of recommended products.
